# BloomTrack 🌸

A pastel gingham wellness dashboard built with HTML, CSS, and JavaScript.

## Features
- `Habit tracking`
- `Sleep tracker with line chart`
- `Mood tracker with donut chart`
- `Local storage persistence`
- `Multi-page structure`
- `Soft aesthetic UI`

## Tech Used
  - HTML
  - CSS
  - JavaScript
  - Chart.js
